# 📚 Library Management System (Java)

This is a simple console-based Library Management System built using Java.  
It allows users to manage books and library members through a menu-driven program.

---

## ✨ Features
- Add new books  
- View all books  
- Search books by title or author  
- Register library members  
- Borrow books  
- Data stored using file handling  

---

## 🛠 Technologies Used
- Java  
- VS Code  
- File Handling  
- OOP Concepts (Classes, Objects, Encapsulation)  
- Collections (ArrayList)  

---

## 📂 Project Structure
LibrarySystem
└── src
└── library
├── Book.java
├── Member.java
├── Library.java
├── FileHandler.java
└── Main.java

---

## ▶ How to Run the Project
1. Open the project folder in VS Code  
2. Open terminal and type:
cd src
javac library/*.java
java library.Main
